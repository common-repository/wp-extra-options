== Description ==

WP Extra Settings gives you control over many different options that are manageable without accessing the code otherwise. This plugin will thus enable you to control almost every aspect of WordPress making a more professional tool out of it.

Security:  
* Hide WP version from meta tags  
* Hide login errors  
* Forbid file editing  
* Disable Live Writer link  
* Disable RSD services  

Dashboard:  
* Remove message from the footer  
* Hide WP version from the footer  
* Disable upgrade notice  
* Disable admin bar  
* Hide WP logo   
* Hide meta boxes   
* Show custom notification      
* Hide Help tab     

Authoring:  
* Show to users only their own posts in the dashboard   
* Allow contributors to upload media files   
* Notify users when their role change     
* <del>Enble Visual Editor in Comments</del>     
* <del>Enble Hidden Buttons in Visual Editor</del>     
* Disable comments in attachment pages    
* Show Pingback/Trackback count in post list  
* Show Attachment count in post list      
* SHow Featured Image Thumbnail in post list     
  
Do you want to raise a bug, propose a change, request for support or simply say hi? You can do it [here](https://bitbucket.org/mirkog/wp-extra-settings/issues?status=new&status=open).

== Installation ==

1. Upload the plugin's folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the settings page under the 'Settings' menu

== Frequently asked questions ==

= I activated many options but it looks like they are not working, what can I do? =
Check that the plugin is enabled under the "General" tab.
